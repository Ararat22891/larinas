# Linux Device Manager

Централизованное приложение для управления Linux устройствами через SSH.

## Возможности

- **Мониторинг устройств**: Отслеживание состояния нескольких Linux машин
  - Использование CPU
  - Использование памяти
  - Использование диска
  - Время работы системы
  - Количество процессов
  - Средняя загрузка системы

- **Файловый менеджер**: Работа с файловой структурой как в Windows
  - Просмотр файлов и папок
  - Навигация по директориям
  - Просмотр информации о файлах (размер, права доступа, владелец)
  - Windows-подобный интерфейс

- **Управление службами**: Полный контроль над systemd службами
  - Просмотр списка всех служб
  - Запуск/остановка служб
  - Перезапуск служб
  - Включение/отключение автозапуска
  - Поиск служб
  - Отображение статуса служб

## Требования

- Flutter SDK 3.9.2 или выше
- Linux устройства с SSH доступом
- Права sudo для управления службами (опционально)
- **Для работы с паролями**: установленный `sshpass`
  - macOS: `brew install hudochenkov/sshpass/sshpass`
  - Linux: `sudo apt-get install sshpass` или `sudo yum install sshpass`
  - **Альтернатива**: Настройте SSH ключи для доступа без пароля

## Использование

### Добавление устройства

1. Нажмите кнопку "+" на главном экране
2. Заполните форму:
   - **Название**: Имя устройства (например, "Мой сервер")
   - **Хост/IP**: IP адрес или доменное имя устройства
   - **Порт**: SSH порт (по умолчанию 22)
   - **Имя пользователя**: SSH пользователь
   - **Пароль**: Пароль для SSH подключения
3. Нажмите "Сохранить"

### Подключение к устройству

1. Выберите устройство из списка
2. Нажмите кнопку подключения в правом верхнем углу
3. После успешного подключения вы увидите три вкладки:
   - **Мониторинг**: Статистика системы
   - **Файлы**: Файловый менеджер
   - **Службы**: Управление службами

### Мониторинг

Экран мониторинга автоматически обновляется каждые 5 секунд. Вы также можете обновить данные вручную, потянув вниз или нажав кнопку обновления.

### Файловый менеджер

- Нажмите на папку для перехода в неё
- Нажмите кнопку "Назад" для возврата в родительскую директорию
- Долгое нажатие на файл/папку открывает контекстное меню
- Нажмите на файл для просмотра информации о нём

### Управление службами

- Используйте поиск для фильтрации служб
- Раскройте службу для просмотра доступных действий
- Доступные действия:
  - **Запустить**: Запустить остановленную службу
  - **Остановить**: Остановить работающую службу
  - **Перезапустить**: Перезапустить службу
  - **Включить/Отключить**: Управление автозапуском

## Безопасность

- Пароли хранятся локально в зашифрованном виде (SharedPreferences)
- SSH соединения используют стандартное шифрование
- Для управления службами требуется sudo права на удалённом устройстве

## Технологии

- **Flutter**: Кроссплатформенный фреймворк
- **Provider**: Управление состоянием
- **SSH**: Библиотека для SSH подключений
- **SharedPreferences**: Локальное хранилище
